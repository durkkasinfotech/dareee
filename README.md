# Aakash Institute Clone - Mobile First Single Page

This is a mobile-first single-page website clone of the Aakash Institute homepage, built with React and Next.js.

## Features

- ✅ Exact replica of the Aakash Institute homepage
- ✅ Mobile-first responsive design
- ✅ Built with Next.js 14 and React
- ✅ Single page application
- ✅ All sections included: Header, Navigation, Hero Banner, Statistics, and Floating Actions

## Getting Started

### Installation

```bash
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

### Build

Build for production:

```bash
npm run build
```

Start production server:

```bash
npm start
```

## Tech Stack

- **Next.js 14** - React framework
- **React 18** - UI library
- **TypeScript** - Type safety
- **CSS** - Mobile-first responsive styling

## Mobile First Design

The website is designed mobile-first, meaning:
- Base styles are optimized for mobile devices
- Responsive breakpoints for tablet (768px) and desktop (1024px)
- Touch-friendly buttons and interactions
- Optimized layout for small screens
